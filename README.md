## Castellano

### ¿Qué es esto?

Colección de presentaciones HTML interactivas para la asignatura de Filosofía de 1.º de Bachillerato (IES Minas BHI, Barakaldo, Euskadi). Cada presentación es un único archivo autocontenido que funciona sin dependencias externas (salvo Google Fonts). Compatible con Safari, Chrome y Firefox, en ordenador y móvil.

### Características principales

- **Reveal progresivo** — los conceptos aparecen uno a uno, mientras los alumnos piensan
- **Estructura de sesión** — Hasiera → Garapena → Itxiera (basada en la pedagogía evidenciada de Héctor Ruiz Martín)
- **Caso central** — cada unidad tiene un caso real como hilo conductor (Barakaldo, Bilbao)
- **Tema claro/oscuro** — adaptable a la luz del aula
- **Impresión** — imprime únicamente la diapositiva activa

### Metodología pedagógica

El diseño aplica los principios de aprendizaje eficaz de Héctor Ruiz Martín (*¿Cómo aprendemos?*, Graó, 2020): recuperación activa, espaciado e interleaving, elaboración interrogativa, y consolidación al cierre de sesión. Los casos centrales locales (Portu kalea, Bilbo) favorecen la transferencia de conceptos filosóficos a situaciones reales y cercanas al alumnado.

## English

### What is this?

A collection of interactive HTML presentations for Philosophy at Year 1 of the Spanish Baccalaureate (Batxilergoa 1), taught entirely in Basque (Model D) at IES Minas BHI, Barakaldo, Basque Country. Each presentation is a single self-contained HTML file requiring no external dependencies except Google Fonts. Compatible with Safari, Chrome, and Firefox on desktop and mobile.

### Pedagogical approach

Built on evidence-based learning principles from Héctor Ruiz Martín (*How We Learn*): retrieval practice, spaced learning, interleaving, elaborative interrogation, and end-of-session consolidation. Each unit features a real-world local case study (a traffic accident in Barakaldo; a housing crisis in Bilbao) as a narrative thread through which philosophical theories are progressively applied.

### Live presentations

Published via GitHub Pages: [https://eukeni-acebal.github.io/filosofia-batx1/](https://eukeni-acebal.github.io/filosofia-batx1/) *(aktibatzear / por activar)*
